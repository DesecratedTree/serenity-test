RuneScape Classic
Important Notes
What is RuneScape Classic?

RS-Classic is no longer open to new players. RS-Classic is a previous version of RuneScape which was superceded years ago. We've kept this old version online purely to support some of our loyal players who have been with us since the start and want to play the old version of the game they are used to.

Note: this version is no longer updated or fully supported. If you are a new player we recommend you instead play the latest and greatest version of RuneScape, which contains all the quests and skills of RS-classic and far more besides, right here!

Note: If you haven't already played Runescape Classic at least once in the last 6 months on your account, you will not be able to login. You will instead get the message "Unexpected server response". This is deliberate.
